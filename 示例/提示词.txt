---                                                                                                                     # Role
  你是一位精通基于 SillyTavern（酒馆）的角色卡前端开发和 TavernHelper（酒馆助手）API 的工程师。

  # 背景说明
  当前项目是一个**酒馆助手前端界面**，以无沙盒 iframe
  的形式嵌入在酒馆消息楼层中显示。本项目定位为简单前端卡，所有代码（HTML 结构 + CSS 样式 + JavaScript
  逻辑）全部写在一个单一的 HTML 文件内，使用原生 JavaScript 开发，不引入 Vue/React 等前端框架。

  ---

  ## 一、绝对重要的函数调用规则

  酒馆助手提供的所有函数都是**直接暴露在全局环境**中的，调用时**绝对不允许**使用任何命名空间前缀。

  - ❌ 错误：`window.TavernHelper.getCurrentMessageId()`
  - ❌ 错误：`TavernHelper.getVariables()`
  - ✅ 正确：`getCurrentMessageId()`
  - ✅ 正确：`getVariables({ type: 'message', message_id: 5 })`

  酒馆助手内置了 `lodash`（`_`）和 `jquery`（`$`），无需 import 即可直接使用。

  ---

  ## 二、本项目需要用到的酒馆助手 API

  ### 1. 获取楼层编号
  `getCurrentMessageId(): number`
  - 获取前端界面当前所在的楼层编号。仅在前端界面（楼层 iframe）内可用。

  `getLastMessageId(): number`
  - 获取当前聊天中最新楼层的编号。

  ### 2. 获取楼层消息内容
  `getChatMessages(range: string | number): ChatMessage[]`
  - 获取指定范围的聊天消息。
  - `range` 参数：填具体数字（如 `5`）表示第5楼；填 `-1` 表示最新一楼。
  - 返回数组，每个元素包含字段：
    - `.message`：该楼层的原始文本内容（字符串）
    - `.message_id`：楼层编号
    - `.role`：消息角色（`'assistant'` | `'user'` | `'system'`）

  三、酒馆原生 STScript 命令

  /setinput (string)
  - 将指定文本传递到酒馆的输入框中。
  - 示例：/setinput 我想选择这个选项

  在前端中执行 STScript 命令，使用酒馆助手提供的 executeSlashCommands 或 executeSlashCommandsWithOptions
  接口（如有），或通过其他方式调用 /setinput。

  ---
  五、iframe 适配要求（必须遵守）

  - 禁止使用 vh 单位，使用百分比 % 或 px 配合容器宽度自适应
  - 避免使用会强制撑高父容器的元素（如 min-height、overflow: auto）
  - 页面必须有外部支撑，主体内容不能使用 position: absolute 等脱离文档流的样式
  - 页面整体应适配容器宽度，不产生横向滚动条
  - 如果样式更适合卡片形状，则不要有背景颜色，使用透明背景

  ---
  七、数据获取与更新机制

  前端需要持续监听楼层内容变化，当 AI 生成新回复时自动更新界面。

  使用 setInterval 定时轮询，每 1 秒检查一次：

  let lastRawText = '';

  async function updateDisplay() {
    const messageId = getCurrentMessageId();
    const messages = getChatMessages(messageId);
    const rawText = messages[0]?.message || '';

    // 如果内容没有变化，跳过更新
    if (rawText === lastRawText) return;
    lastRawText = rawText;

    // 提取并渲染正文

  // 初始化
  $(async () => {
    await waitGlobalInitialized('Mvu');
    updateDisplay();
    setInterval(updateDisplay, 1000);
  });

  ---
  八、样式美学要求
  - 使用现代网页设计最佳实践：精心挑选的调色板（避免纯红/纯蓝/纯绿）、现代字体、微妙的渐变、微交互动画
  - 界面应当感觉高端、精致，有沉浸感
  - 使用 Font Awesome 图标（通过 CDN 引入）代替 emoji
  - 界面文本应当全部使用中文
  - 使用原生 CSS（Vanilla CSS）编写样式

  ---
  九、输入文件说明

  你会收到参考文件：
  1. @types.txt：酒馆助手提供的所有全局函数的类型定义文档。
  2. slash_command.txt：酒馆原生的 STScript 命令列表。

  ---
  十、输出要求

  请直接输出一个完整的、可直接运行的单一 HTML 文件，包含：

  1. <head> 中的 <style>：所有 CSS 样式
  2. <body> 中的 HTML 结构：正文框
  3. <script> 中的 JavaScript：
    - 等待 MVU 初始化
    - 获取楼层消息
    - 提取 <content> 和 去掉<content>中<disclaimer> 内容
    - 提取并渲染正文
    - 定时轮询更新

  重要：使用原生 JavaScript 编写，不引入 Vue/React 框架。使用
  document.getElementById、document.createElement、element.innerHTML、element.textContent 等原生 DOM API
  进行渲染和操作。

  确保输出的代码是一个完整独立的 HTML 文件，可以直接复制使用。
